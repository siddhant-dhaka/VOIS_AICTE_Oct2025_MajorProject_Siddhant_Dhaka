🎬 Netflix Data Analysis

Unlocking insights from Netflix’s content library using data analysis and visualization. This project explores trends in movies and TV shows to understand content growth, genre popularity, and audience preferences.

📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on Netflix data to extract meaningful insights about:

Content distribution (Movies vs TV Shows)
Genre trends and popularity
Release patterns over time
Country-wise content production
Ratings and audience preferences
🎯 Key Objectives

📈 Content Trends
Analyze how Netflix content has grown over the years
Compare Movies vs TV Shows distribution

🎭 Genre Analysis
Identify most popular genres
Understand content diversity

🌍 Regional Insights
Explore country-wise content production
Identify top contributing countries

⭐ Ratings & Preferences
Study ratings distribution
Understand what type of content performs better

🛠 Tech Stack
Python 3.8+
Libraries Used:
pandas – Data manipulation
matplotlib – Data visualization
seaborn – Statistical plots
numpy – Numerical operations

📊 Key Insights
📌 Movies dominate the Netflix catalog compared to TV Shows
📌 Significant growth in content after 2015
📌 Certain genres consistently outperform others
📌 Content production is concentrated in a few countries
